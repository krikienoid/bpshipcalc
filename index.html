FORMULAS

SPECIAL MODIFIERS

special modifiers for most specials are stacked using this formula

a * (1 + b) = c

a = modifier value 1
b = modifier value 2
c = stacked modifier value

DEFENSE BONUS

a * (1 - b) = c

a = defense bonus 1
b = defense bonus 2
c = stacked defense bonus

WEIGHT

calculating total weight

hidden_weight = FLOOR(hull_armor / 4)

hull_armor = base armor points of hull

WP(p, wm) = FLOOR( FLOOR(p_weight * mYL) * wm)

p = ship part
wm = weight modifer
p_weight = weight of ship part, p
mYL = shipyard level modifier (mYL = 1 if level 1, mYL = 0.85 if level 2)

total_weight = FLOOR(
(WP(w1, mW) + WP(w2, mW) + ... + WP(weaponn, mW) +
WP(a1, mA) + WP(a2, mA) + ... + WP(armorn, mA) +
WP(tm1, mTM) + WP(tm2, mTM) + ... + WP(tmn, mTM) +
FLOOR(hidden_weight * mA) - hidden_weight ) * mT
)

w = weapon
a = armor
tm = tactical module

mT = Total Weight modifier
mW = Weapon Weight modifier
mA = Armor Weight modifier
mTM = Tactical Module Weight modifier

DPS

damage per second, t

t = (a * d)/(r + s * f)

a = Accuracy
d = Damage
r = Reload
s = Salvo
f = firing speed (usually around 0.1)
